MCP Tools for Agents

        

        

          MCP (Model Context Protocol) 도구는 AI 에이전트가 Mories의 기억을 직접 조작할 수 있게 해주는 JSON-RPC 인터페이스입니다.

          엔드포인트: POST /api/mcp | 프로토콜: JSON-RPC 2.0
        

      


      

        
🔧 Available MCP Tools


        

          
mories_search

          
지식 그래프에서 벡터 유사도 + 키워드 기반으로 기억을 검색합니다. 검색된 기억은 자동으로 Retrieval Boost됩니다.

          

            Params: query (string), graph_id (string), limit (int, default: 10)
          

        


        

          
mories_ingest

          
텍스트/문서를 지식 그래프에 인제스트합니다. 엔티티 추출 → 관계 구축 → STM 승격 파이프라인을 자동 실행합니다.

          

            Params: content (string), source (string), graph_id (string), auto_relate (bool)
          

        


        

          
mories_profile

          
특정 주제에 대한 종합 프로필을 구축합니다. 관련 엔티티, 관계, 사실을 수집하여 구조화된 프로필을 반환합니다.

          

            Params: subject (string), graph_id (string), depth (int, default: 2)
          

        


        

          
mories_graph_query

          
Cypher 쿼리를 직접 실행합니다 (읽기 전용). Neo4j 그래프 데이터를 직접 탐색할 수 있습니다.

          

            Params: cypher (string), params (dict), limit (int, default: 50)
          

        


        

          
mories_stream

          
스트리밍 데이터 소스를 등록하거나 관리합니다. 실시간 데이터를 지속적으로 지식 그래프에 인제스트할 수 있습니다.

          

            Params: action (create|stop|list), stream_id (string), query (string)
          

        


        

          
mories_harness_list

          
모든 하네스(진화형 프로세스) 패턴 목록을 조회합니다. 도메인 및 에이전트 ID로 필터링 가능합니다.

          

            Params: domain (string, optional), agent_id (string, optional), include_low_success (bool, optional)
          

        


        

          
mories_harness_overview

          
하네스 패턴에 대한 누적 통계가 포함된 대시보드 개요 데이터를 반환합니다.

          

            Params: None
          

        


        

          
mories_harness_record

          
새로운 하네스 패턴을 기록합니다. 실패-조건부-회복 메커니즘 등을 지정할 수 있습니다.

          

            Params: domain (string), trigger (string), tool_chain (list), description (string, optional), scope (string, optional)
          

        


        

          
mories_harness_detail

          
특정 식별자를 사용하는 하네스 패턴의 세부 정보(데이터 흐름, 진화 내역, 조건 등)를 조회합니다.

          

            Params: uuid (string, required)
          

        


        

          
mories_harness_execute

          
하네스 패턴 실행 결과를 기록하여 작동 내역, 성공률 및 통계를 누적 업데이트합니다.

          

            Params: uuid (string, required), success (bool, required), execution_time_ms (int, optional), result_summary (string, optional), new_tool_chain (list, optional)
          

        


        

          
mories_harness_evolve

          
기존 하네스 패턴을 진화시켜 새로운 도구 체인과 실행 내역을 가진 신규 버전을 생성합니다.

          

            Params: uuid (string, required), new_tool_chain (list, required), reason (string, required)
          

        


        

          
mories_harness_recommend

          
키워드 및 관련도를 바탕으로 AI 추천 기반의 하네스 프로세스 패턴을 검색합니다.

          

            Params: query (string, required), domain (string, optional), cross_domain (bool, optional), limit (int, optional)
          

        


        

          
mories_harness_rollback

          
하네스 패턴의 에볼루션 히스토리를 기반으로 과거의 특정 버전으로 수동 롤백합니다.

          

            Params: uuid (string, required), to_version (int, required)
